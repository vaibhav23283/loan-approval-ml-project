# Loan Approval Prediction (ML Project)

This project predicts whether a loan will be approved based on applicant details using machine learning.

## Models Used

* Logistic Regression (Best Model)
* Decision Tree
* Random Forest

## Results

* Logistic Regression Accuracy: ~83.7%
* High Recall (~0.98) ensuring most approvals are correctly identified

## Key Learnings

* Data preprocessing (handling missing values, encoding)
* Model comparison and evaluation
* Understanding precision, recall, and F1 score

## Conclusion

Logistic Regression performed best, showing strong generalization and balanced performance.
